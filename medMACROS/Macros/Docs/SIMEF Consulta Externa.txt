-- Sección Datos Derechohabiente
1.1 Nombre
	Muestra el nombre del expediente actualmente abierto en SIMEF. Al presionar CTRL + F5 mientras se encuentra abierto el expediente del paciente en SIMEF, este recuadro se llenará automáticamente.
1.2 Cédula
	Se puede escribir la cédula del paciente en este campo, al presionar "Abrir en SIMEF" se abrirá este expediente.
	Los botonoes RFC y RFC/Tipo se pueden usar para pegar rápidamente la cédula del paciente en algún lugar donde se necesite.
	Nota: por error de diseño del SIMEF, si el tipo de derechohabiente termina en un número distinto a 0, deberá de seleccionarse manualmente el tipo de derechohabiente.
1.3 Pegar datos
	Aquí se pueden presionar los botones con su nombre correspondiente para pegar los datos en el campo de texto en el que se esté escribiendo.
2 -- Sección Pantalla principal
2.1 Abrir SIMEF
	Si se encuentran guardados los datos, este botón abrirá automáticamente el SIMEF e ingresará a su sesión.
2.2 Informe diario
	Se abrirá la pantalla de informe diario de SIMEF.
2.3 Nueva consulta
	Se abrirá una nueva pantalla de consulta en SIMEF.
3 -- Sección Pantalla de consulta
3.1 Abrir historial
	Abrirá el historial de consultas del paciente actual.
3.2 Siguiente consulta 
	Mientras se encuentra en alguna pantalla de una consulta previa, presionar este botón pasará a la nota anterior a la que se está viendo.
3.3 Cerrar historial
	Mientras se encuentra en la pantalla de la consulta, presionar este botón cierra la ventana, y regresa a la pantalla principal.
3.4 Escribir consulta
	En ocasiones, la pérdida de conexión con el sistema ocasiona que se cierre SIMEF, borrando la nota que se estaba escribiendo.
	Esta función genera una ventana en donde se puede escribir la nota de la consulta junto con los diagnósticos y signos vitales. 
	Al terminar, se coloca el cursor en el cuadro de escribir consulta en el SIMEF, se presiona el botón "enviar a SIMEF" de la ventana, y los datos se copiarán al SIMEF para poder guardar la consulta. 
	La ventana posee además la opción de guardar lo que se haya escrito como un archivo de texto, o abrir archivos de texto previamente existentes para poder mandarlos al SIMEF.
